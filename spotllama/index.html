A Speech+Text Generative Language Model


We introduce Spot-llama, a multimodal language model that freely mixes text and speech, and is trained jointly with tokens extracted from both speech and text corpora. The speech tokens are based on speech units (HuBERT based), and the text tokens are BPE based. They are concatenated as a single set of tokens, and trained using a word-level interleaving method using a small speech-text parallel corpus. The resulting model displays both the semantic abilities of text models and the expressive abilities of speech models. We demonstrate that Spot-llama is able to learn new tasks in a few shot fashion across modalities (ASR, TTS, speech classification).
